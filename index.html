NE Watershed Network

Nebraska Watershed Citizen Science Project

Nebraska Watershed Network Application is a citizen science project application built to help volunteers across Nebraska to collect water samples and upload the minerals content to information to a central database which can be used by researchers at the University to conduct Research on Ground Water.

This application made a tedious process super easy. Prior to this applications use, volunteers had to call the co-ordinators to get the test kits and then mail the test results which tooks a very long time. Now volunteers can request the test-kits by the press of the button and upload the results within minutes from their mobile. It also points the results on a map to see trends in mineral content across Nebraska.

This project code base is being used in Idaho to bulid their version of the application as well.
